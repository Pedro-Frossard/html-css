Outros tipos de formatação ou pre-formatação

Tag code

Sem tag code, com tag code o espaço entre as letras fica igual

Tag pre

A tag pre faz as quebras de linha e identação igual como esta escrido no codigo, por exemplo

        
num = int(input('Digite um número: '))
if num%2 == 0:
    print('Numero é par')
else:
    print('Numero é impar')
print('Fim')
